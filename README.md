# Freedom Is Coming

一个具有赛博朋克/黑客美学风格的单页面网站，展示"自由即将降临"的主题概念。

## ✨ 特性

- **3D 黑洞视觉效果** - 使用 Three.js 实现的吸积盘粒子系统，模拟黑洞事件视界
- **文本解密动画** - 黑客风格的文本加密/解密动态效果
- **CRT 屏幕特效** - 复古扫描线和暗角效果
- **HUD 界面元素** - 科幻风格的状态显示界面
- **鼠标视差效果** - 摄像机跟随鼠标移动的平滑视差
- **交互爆发效果** - 点击按钮触发粒子加速扩散动画

## 🛠️ 技术栈

- HTML5 / CSS3
- JavaScript (ES6+)
- [Three.js](https://threejs.org/) (r128) - 3D 渲染引擎
- Canvas API - 粒子贴图生成

## 🚀 使用方法

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件
3. 移动鼠标体验视差效果
4. 点击 "INITIATE OVERRIDE" 按钮触发交互效果

无需任何构建工具或依赖安装，开箱即用。

## 🎨 视觉设计

- **配色方案**: 深黑背景 (#030303) + 纯白主色 + 红色警告色 (#ff2a2a)
- **字体**: 
  - 标题: Syncopate (未来科技感)
  - 正文: Space Mono (等宽编程风格)
- **粒子系统**: 45,000 个粒子构成的吸积盘，从白色核心渐变到红色再到灰色边缘

## 🌟 亮点功能

### 黑洞吸积盘
粒子围绕中心黑洞旋转，遵循类开普勒定律 - 越靠近中心旋转越快。

### 文本解密效果
使用自定义 TextScramble 类实现字符逐个解密的黑客风格动画。

### 交互爆发
点击按钮后，粒子加速并向外扩散，模拟能量爆发的视觉效果。

## 📝 开发说明

项目采用单文件架构，所有 CSS 和 JavaScript 都内联在 `index.html` 中，便于快速部署和分享。

3D 渲染使用 Three.js 的 Points 粒子系统，通过 Canvas 动态生成发光粒子贴图，避免外部资源依赖。
